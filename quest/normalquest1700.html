一般クエスト　Lv1675～9999


[連作クエスト](ウモス前提クエ)：2000Lv以降のクエスト進行に必要。スフィア・通常パワキ込みで合計4500億EXP程度。
衝突 -> 誘き寄せられた光 -> 隠された聖なる地 -> 失踪した司祭達 -> 壊滅 -> 盗まれた聖物 -> 苦悩と激動 -> 原始の輪 -> 闇の革命 -> ウモスクェグマイヤーの深い所
原始の輪・闇の革命では秘密ダンジョン原始の輪を進行します。
前提としてウポス連作クエスト（オナクシヌ討伐して戻るところまで）と、ある異邦人の完了が必要です。



ウモスクェグマイヤーを中心に回るクエスト。
2000Lv以降のクエストを進行するには、ウモス前提クエに加えて、ウポス連作クエスト（オナクシヌ討伐して戻るところまで）とある異邦人を完了している必要があります。
進行中は指定されたタイミング以外で秘密ダンジョン原始の輪に入場できないため注意。

移管前から1500Lv以上になっており、ウポスクエを完了している場合は初手でセルーテのハイドネルと話せない不具合がある可能性があります。
その場合はセルーテのハイドネルと話せるよう以下のような連絡をサポートセンターに送ってください。
