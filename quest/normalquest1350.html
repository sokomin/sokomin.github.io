一般クエスト　Lv1350～9999


[連作クエスト](第2旅程)：スフィアパワキ込みで合計2510億EXP、妖精の翼のカケラを獲得可能。
派遣のお知らせ -> 頑強な妨害 -> 密かに調査するために -> 追い迫る影 -> 二つ目の旅路
ヤティカヌ連作クエストの「その茎を切れ」を完了することで受諾可能になります。
アップデートによって追加される新規1550-1675Lvのマップと何らかの関係があるクエスト。

[連作クエスト](沼地の向こうに)：スフィアパワキ込みで合計2600億EXP、ウポス補助武器を獲得可能。
沼地の向こうに -> ウポスマイヤー北部 -> ウポスマイヤー調査 -> 周辺モンスター退治 -> ウモスクェグマイヤーに行く道
