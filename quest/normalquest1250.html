一般クエスト　Lv1250～1350


[連作クエスト](ヤティカヌクエ)：スフィアパワキ込みで合計390億EXP獲得可能。
未知へ向けて -> 不安な気配 -> 招かれざる者 -> あくまでも安全のために -> 怪しい音
-> 助けの手 -> 部族の掟 -> 妖精の試験 -> 目覚めた災い -> 魔花アリドネ -> その茎を切れ
「未知へ向けて」はヤティカヌフィールド入場に必要。
ガレリオン野営地に直通するためには招かれざる者をクリアする必要あり。
以降のクエストは任意ですが、その茎を切れを完了することでヤティカヌウィークリークエストを受諾可能になります。

これらのクエストを進行するためには、「探検隊ガレリオン」関連クエストを4つとも完了する必要があります。
公式ページにて詳細が記載されています。
