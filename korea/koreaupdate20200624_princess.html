韓国 2020/6/24 プリンセススキル改変

[重要！] 韓国公式の情報を基にしています。
誤訳や韓国独自仕様の可能性もありますので、予めご了承下さい。

[姫 ] （PvE / PvP）< 一般スキル>
● [ 病気投げ] PvP ダメージとダメージ属性が変更され、スキル効果が変更されます。
●[ポーション投げ]スキル効果が変更/追加され、攻撃範囲の数値が変更されます。
●[ ポーション投げ] すぐに回復数値が変更され、知識ステータスに影響を受けます。
● [ 薬投げ] スキル名とスキル効果が変更され、パッシブスキルに変更されます。
● [ フラワーシャワー] スキル名とスキル効果が変更されます。
● [ フラワーシャワー]消費/ 獲得CP 数値とすべての抵抗力の低下数値、攻撃範囲の数値が変更されます。
● [ キャンディーシャワー] スキル名とスキル効果が追加されます。
● [ キャンディーシャワー] 混乱発動確率と持続時間の数値が変更され、攻撃範囲が変更されます。

[ 障害の女王] （PvE / PvP）< 元スキル>
● [ 魔法のスリング] パッシブスキルの効果が変更されます。
● [ 病気すいとん] スキル名とスキル効果が変更されます。
● [ 病気すいとん] PvP ダメージ数値とボトルが爆発ダメージの数値が変更されます。
● [ 即効薬投げサイクル] スキル名とスキル効果が変更され、パッシブスキルに変更されます。
● [ フラワーブーケシャワー] スキル名とスキル効果が変更されます。
● [ フラワーブーケシャワー] 消費/ 獲得CP数値とすべての抵抗力の低下数値、攻撃範囲の数値が変更されます。
