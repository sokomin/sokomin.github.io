10月29日（水）より行われるRS PASSシーズン18をご案内し、
冒険者の皆様はゲーム利用に参考にしてください。
RS PASSシーズン18のご案内
  • 期間： 2025年10月29日(水)メンテ後〜2025年12月31日(水)メンテ前まで
  • 内容 : 指定されたコンテンツを進めて経験値を積んでランクを上げて報酬を受ける RS PASSシーズン18 が進行されます。
  • RS PASSシーズン18は、 ゲーム内のミニマップの左側のRS PASSアイコンとして参加可能です。
    ＊アカウント内100レベル以上のキャラクターでのみ参加可能です。

    ＊RS PASS報酬はシーズン期間内に直接クリックして受け取る必要があり、シーズン終了後は報酬を受けることができません。


    RS PASS経験値取得方法
    • 指定されたコンテンツを進行してRS PASS 経験値を獲得できます。
    • RS PASS経験値は 1日最大130経験値 まで獲得が可能です。

      RS PASSシーズン18アップグレードと報酬ガイド
      • RS PASS経験値を獲得して、上昇したランクに応じてRS PASS報酬を獲得でき、 RS PASSアップグレード券 を購入するとランク別に RS PASSプレミアム報酬 を追加で獲得できます。

      • RS PASSの報酬は、アカウントごとに一度だけ獲得できます。
      • プレミアム報酬は、買い物かごで RS PASSアップグレード権アイテムを適用した サーバーにのみ適用されます。
        （アカウントごとに1回の購入制限で同じアカウントを使用する他のサーバーではプレミアム報酬を受け取ることはできません。)



        ＊使用直後、 現在のRS PASS Rankで+10 Rank が適用されます。

        例） 10 Rank から RS PASS アップグレード券を使用する場合 20 Rank で適用
        ＊ RS PASS アップグレード券はアカウントごとに 1 個のみ購入が可能です。


        ＊買い物かごから在庫にアイテムを移動する場合、申請の撤回を進めることはできません。
