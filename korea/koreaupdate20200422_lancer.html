韓国 2020/4/22 ランサースキル改変

[重要！] 韓国公式の情報を基にしています。
誤訳や韓国独自仕様の可能性もありますので、予めご了承下さい。



[ ランサー] （PvE / PvP）< 一般スキル>
● [ ラピッドスティンガー] 物理攻撃力数値が変更されます。
● [ 槍投げ] スキル効果が変更されます。
● [ イリュージョンスピア] スキルの名前が変更され、武器低下効果と命中率/回避率低下効果が削除されます。
● [ イリュージョンスフィア] 硬直効果と持続時間が追加され、クールタイムが追加されます。
● [ ジャベリンテンペスト] 覚醒スキルで指定され、風ダメージが削除されます。
● [ ジャベリンテンペスト] 攻撃回数が追加され、攻撃範囲と射程距離が変更されます。
● [ サイドステップ] 回避距離が削除されます。
● [ スピンアライジング] 硬直効果とノックバック効果が削除され、攻撃範囲が追加されます。
● [ スピンアライジング] 獲得CP 数値と物理ダメ、発動確率が変更されます。
● [ アーマーディスアセンブラ] 防御力の低下％が変更され、発動条件が変更されます。
● [ ダミーステップ] スキルの名前が変更され、発動確率と回避距離、ダミー生成効果が削除されます。
● [ ダミーステップ] 消費CP 数値が変更され、光属性追加ダメージが追加されます。
● [ ワールランニング] 硬直効果が削除されます。
● [ ワーリングアサルト] スキルの名前が変更され、ノックバック効果と移動速度増加効果、ブロック無視効果が削除されます。
● [ ワーリングアサルト] 攻撃回数効果が追加され、攻撃範囲が追加されます。
● [ マルチプルツイスター] ノックバック効果と移動速度増加効果、ブロック無視効果、同時攻撃の効果が削除されます。
● [ マルチプルツイスター] スキルの名前が変更され、攻撃回数と最小射程距離が追加されます。
● [ グランドシェーカー]硬直発動確率と硬直時間が変更されます。
● [ ラジアルアーク] 風/ 光ダメージ値と攻撃範囲、攻撃回数が変更される予定です。
● [ ラジアルアーク]射程距離が追加され、段階別ダメージ効果が削除されます
● [ ファイアアンドアイス] 火/ 水ダメージ数値が変更され、攻撃回数が追加されます。
● [ ファイアアンドアイス] 段階別ダメージ効果が削除されます。
● [ エントラップメントピアシング] 物理攻撃力数値が変更され、同時攻撃回数最大値が変更されます。
● [ オーサムフォートレス] 同時攻撃回数が変更され、攻撃回数と攻撃範囲が追加されます。
● [ 信念] 持続時間が変更されます。


[ヴァンガードキャバリエ ] （PvE / PvP 共通）<覚醒スキル>
●[連続攻撃]パッシブ効果が変更されます。
● [ パラライズスティンガー] 物理攻撃力数値が変更され、致命打発動確率が削除されます。
● [ パラライズスティンガー] 攻撃速度増加効果が追加されます。
● [ エントラップメントインペール] 物理攻撃力の数値と同時攻撃回数最大値が変更されます。


[エレメンタルランサー ] （PvE / PvP 共通）<覚醒スキル>
●[ラジアルアーク]風/光ダメージ数値と攻撃範囲が変更され、攻撃回数と射程距離が追加されます。
● [ ラジアルアーク] のステップ星ダメージが削除されます。
● [ ブレイズ・アンド・ブリザード] 火/ 水ダメージ数値と攻撃回数が変更されます。
● [ ブレイズ・アンド・ブリザード] 近接時に追加のスキル効果が追加され、段階別ダメージが削除されます。
● [ 確固たる信念] 持続時間が変更されます。


[ロージングヴァルキリー] （PvE / PvP 共通）<覚醒スキル>
● [ 巧妙な戦術] パッシブ効果が変更されます。
● [ インボルソリューションツイスター] スキルの名前と攻撃回数、攻撃範囲が変更されます。
● [ インボルションツイスター] ノックバック効果が同時打撃効果が削除され、元のパッシブ強化効果と最小射程距離が追加されます。
● [ マエルレームアサルト] スキルの名前が変更され、武器攻撃範囲補正効果とノックバック効果が削除されます。
● [ マエルレームアサルト] 移動速度増加効果と射程距離が削除され、攻撃回数が追加されます。
● [ ジャベリンストーム] 新規スキルが追加されます。


[ランサー] （PvP）<一般スキル>
● [ スピンアライジング] PvEと同じように適用される予定です。
● [ ファイアアンドアイス] PvE と同じように適用される予定です。
● [ マルチプルツイスター] PvE と同じように適用される予定です。
● [ ワーリングアサルト] PvE と同じように適用される予定です。
● [ グランドシェーカー] PvE と同じように適用される予定です。
● [ イリュージョンスフィア] PvE と同じように適用される予定です。
● [ ワールランニング] と[ サプライジングレイド] がクールタイムを共有しない。

[ ランサー] （PvP）< 覚醒スキル>
●[ブレイズ・アンド・ブリザード] PvEと同じように適用される予定です。
● [ ラピッドスティンガー] PvE と同じように適用される予定です。
● [ マエルレームアサルト] PvE と同じように適用される予定です。
● [ ジャベリンストーム] PvP 進行時スキル効果が変更されて適用される予定です。
