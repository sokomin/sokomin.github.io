## 見やすくする作法
- VSCode前提ね

- `<b>\n(.*?)<font>`
```
  <br><b>
$1<font>
```
※2個<br>でもいいかも

- `。(.?)`

```
。<br>$1
```

- `                <table id="table10" style="min-width: 960px; max-width: 960px;">`

```
                <table id="table10" style="min-width: 960px; max-width: 1200px;">
                    <colgroup>
                        <col span="1" width="135">
                    </colgroup>
```
