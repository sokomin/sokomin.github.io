移動速度(%)を入力すると、走り・絨毯・ギルド戦での 1frame あたりの進行距離と、1 座標進むのに必要な frame 数を計算します。

計算式

● 1frame に進む座標距離は移動速度(%)から以下で算出。
　走 り の 場 合 ： { 0.0025 × [ 移動速度(%) ] + 0.25 }
　絨 毯 の 場 合 ： { 0.0075 × [ 移動速度(%) ] + 0.75 }
　Ｇ Ｖ の 場 合 ： { 0.0005 × [ 移動速度(%) ] + 0.2 }
● ただし 1frame に 1 座標距離が最速 (上限)。
● 絨毯は「移動速度オプションや補正の恩恵が通常の3倍」が係数3倍として既に式に内包されているため、ここではキャラ素の移動速度(%)をそのまま入力する。
● キャップ到達移動速度: 走り 300%、絨毯 約33.33%、ギルド戦 1600%。
● ウィンドジャンパーは移動速度増加無しのカーペットと考えるとわかりやすいです。
● 秘密ダンジョン内での移動速度はギルド戦での計算式に準じます。

● 参考: 魔法のカーペット
